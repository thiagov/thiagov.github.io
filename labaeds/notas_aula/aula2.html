A qual caractere certo inteiro corresponde?

Nesse exemplo vamos analisar um programa que deve ter o seguinte funcionamento: dado um número inteiro qualquer, fornecido pelo usuário, devemos escrever na tela qual é o caractere correspondente a tal número na tabela ASCII. O programa em C é mostrado a seguir:

#include <stdio.h> int main(int argc, char *argv[]) { int x; printf("Entre com um numero inteiro: "); scanf("%d", &x); printf("O caractere correspondente ao numero digitado eh: %c\n", x); return 0; }

Se você compilar e executar o programa acima, obterá uma saída do seguinte formato: 

    
        Entre com um numero inteiro: 65
        O caractere correspondente ao numero digitado eh: A

No exemplo mostrado acima, o usuário entrou com o número 65. O programa mostrou, corretamente, que o caractere correspondente a esse número na tabela ASCII é o caractere A. Como esse programa funciona? Vamos analisar as linhas importantes desse código.

Linha 6
Nessa linha estamos somente lendo um valor que o usuário deve fornecer, usando a função scanf(). Como especificado, o usuário deve entrar com um número inteiro. Dessa forma, usamos o "%d" no scanf() para ler o valor. Também passamos para o scanf() a variável que armazenará o valor digitado: nesse caso a variável x. Devemos lembrar que o nome da variável deve vir precedido de um & quando estamos lendo um tipo básico.
Linha 8
Nessa linha fazemos a impressão do caractere correspondente ao número inteiro armazenado em nossa variável x. O procedimento é bastante simples: basta imprimir a variável x (um número inteiro) utilizando %c ao invés de %d! Na linguagem C, caracteres são mapeados por números da tabela ASCII. Dessa forma, um valor inteiro é automaticamente convertido para o caractere correspondente da tabela ao pedirmos a impressão com %c.

Nesse exemplo queremos fazer um programa que, dado certo número de segundos, nos informa a hora no formato horas:minutos:segundos. O programa é mostrado a seguir:

#include <stdio.h> int main(int argc, char *argv[]) { int horas; int minutos; int segundos; printf("Entre com a quantidade de segundos: "); scanf("%d", &segundos); minutos = segundos/60; segundos = segundos%60; horas = minutos/60; minutos = minutos%60; printf("%d horas, %d minutos e %d segundos!\n", horas, minutos, segundos); return 0; }

Se você compilar e executar o programa acima, obterá uma saída do seguinte formato: 

    
        Entre com a quantidade de segundos: 600012
